# created by cargo setup

## create new app
```
cargo new my-app
```

## check version
```
rustc --version
```

## check cargo version
```
cargo --version
```

## build program
```
cargo run
```

## run program
```
rustc main.rs
```

## Add dependencies
inseide cargo.toml file, then run
```
cargo build
```
